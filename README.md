# Nome do projeto: NextStep

## INSTALAÇÃO DO PROJETO PASSO A PASSO:

⦁	Para habilitar a instalação do programa, será necessário primeiro assinar o certificado digital dele e depois executar o instalador

COMO ASSINAR O CERTIFICADO:

1.	Dentro da pasta do instalador (appProgComp_1.0.0.0_Test) você devera encontrar um arquivo chamado "appProgComp_1.0.0.0_x64.cer", execute-o e espere ele abrir
2.	Quando ele abrir, clique no botão escrito "Instalar Certificado"
3.	Após isso, ele ira abrir outra página de importação de certificado, você deverá selecionar a opção "Máquina Local" e clicar no botão avançar (Obs: caso o sistema peça permissão de administrador você deverá permitir)
4.	Após você escolher importar o certificado na máquina local, outra página irá abrir com duas opções, NÃO escolha a opção de selecionar automaticamente o repositório de certificados conforme o tipo de certificado, Você deve clicar na opção que permite você escolher o repositório em que o certificado vai ser colocado
5.	Após clicar na opção que permite você escolher o repositório, clique no botão "Procurar", ao lado do campo de texto, e selecione a pasta "Autoridades de Certificação Raiz Confiáveis", depois clique no botão "Ok"
6.	Com o repositório de certificado já escolhido, pode clicar no botão "Avançar", outra página vai abrir para concluir o processo de importação, pode clicar no botão "Concluir"
7.	Se tudo ocorreu bem, um aviso avisando que "a importação obteve êxito" vai aparecer, pode fechar ele

⦁	PRONTO, Agora que você importou o certificado digital na sua máquina, o programa poderá ser instalado na sua máquina, segue abaixo o passo a passo para fazer sua instalação

COMO INSTALAR O PROGRAMA:

1.	Dentro da pasta appProgComp_1.0.0.0_Test você deve achar um arquivo de formato .msix com o seguinte nome: "appProgComp_1.0.0.0_x64.msix", clique duas vezes nele para executar
2.	Uma página de instalação do Programa NextStep vai abrir, pode clicar no botão "Instalar" e se quiser, ativar a opção "Iniciar quando pronto", após esse passo o programa será instalado na máquina e poderá ser executado.

⦁	Após todos esses passos, o programa será instalado e deve rodar sem problemas
